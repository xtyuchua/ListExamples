 Overview

This project demonstrates how to use the Java List interface with an ArrayList implementation. It showcases common operations such as adding, removing, searching, sorting, and iterating through elements.

 Features

The program performs the following operations on a list of fruits:

 Add elements to the list

 Insert elements at a specific position

 Access elements using index

 Check list size

 Search for elements (contains, indexOf)

 Remove elements (by value and index)

 Iterate using:

Enhanced for-loop

Traditional for-loop

 Sort elements:

Alphabetically

Reverse order

By string length
